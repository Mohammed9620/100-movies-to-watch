# 100 Movies to Watch (Web Scraper)

## Description
A Python-based web scraping application that automatically extracts a list of the top 100 movies to watch.

## Key Features
* **Automated Scraping**: Uses **BeautifulSoup4** to parse HTML and extract movie titles
* **Data Cleaning**: Implements string manipulation to clean titles and formatting
* **UTF-8 Handling**: Ensures special characters and symbols are preserved using UTF-8
* **Ordered Output**: Reverses the original list to provide a chronological 1-100 order

## Technologies Used
* **Python 3.x**
* **BeautifulSoup4**: For robust HTML parsing.
* **Requests**: For managing HTTP connections to the web archive.

## Installation & Usage
1. **Clone the repository**:
```bash
git clone https://github.com/Mohammed9620/100-movies-to-watch.git
```

2. **Install dependencies**:
```bash
pip install beautifulsoup4 requests
```

3. **Run the script**:
```bash
python main.py
```
